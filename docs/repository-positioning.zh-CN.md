# 仓库定位说明

## howeverpromptfoo 与上游仓库的角色边界

### 上游社区版

- 面向广泛社区需求
- 强调生态兼容和功能覆盖
- 版本节奏以官方发布为准

### howeverpromptfoo 个人发行版

- 面向个人/团队工程落地
- 强调本地化配置治理与中文文档可交接性
- 保持与上游可同步，但允许增加治理脚本和工程模板

## 仓库描述模板

`HoweverPromptfoo：面向企业场景的 LLM 评测与红队测试框架，支持 Prompt 评估、模型对比、安全测试与 CI 集成。`
